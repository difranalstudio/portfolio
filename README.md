# Portfólio Pessoal — Diogo Alves

Portfólio desenvolvido com Django para apresentar meus projetos, competências técnicas e informações de contato.

## Sobre o projeto

Este projeto foi desenvolvido com foco em apresentar minha trajetória como desenvolvedor Python, reunindo:

- Projetos pessoais
- Competências técnicas
- Experiência com dados, automação e IA
- Links para GitHub e contato profissional

## Funcionalidades

- Página inicial personalizada
- Seção de competências por categoria
- Cards de projetos dinâmicos
- Página de detalhes de cada projeto
- Painel administrativo com Django Admin
- Upload de imagens para projetos
- Layout responsivo
